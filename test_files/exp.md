# 玩转蓝晒：解锁成像效果的秘密

### ——以曝光时间、原料配比、显影剂为例

组员：周圣烨 徐皓哲

辅导老师：谢思倩

> **前言**
>
> 快门按下一瞬间得到的照片不仅记录着当下我们的生活，还是留存人类历史的发展重要载体。翻开初中历史课本，从五四运动高扬的旗帜到工业革命的蒸汽机，黑白照片虽模糊不清却让历史鲜活起来。接触了化学这门学科后，我们在一次资料查阅中惊喜地发现，在只有“黑白”的19世纪中，竟有一抹浪漫的普鲁士蓝——蓝晒。作为“蓝图”（Blueprint）一词的直接来源，蓝晒为近现代工业发展做出不可磨灭的贡献，并在逐步发展中被赋予艺术价值。本实验，将从化学角度探秘蓝晒制作的奥秘。

## 一、研究目的

在了解蓝晒发展历程和制作方法的过程中，我们发现不同的用途、工艺、环境以及制作者下诞生的蓝晒各有不同的鲜明特色。有的色彩浓郁，有的清透柔和，有的层次分明。那在我们自己的艺术创作中，该如何调出独具一格、极具诗意的宝蓝呢？为解决这个问题，我们决定用初中所学的化学知识，通过严谨的对比实验，从曝光时间、原料体积比、显影剂选择、负片制作四个角度探索蓝晒成像的秘密。同时，从中学生的独特视角，以创新设计，打造具有时代烙印的艺术作品。

## 二、研究过程

### （一）文献调查：从历史中寻找线索

![](media/image1.jpeg){width="0.883176946631671in" height="1.1516622922134734in"} ![](media/image2.jpeg){width="1.4015179352580927in" height="2.032034120734908in"}

19世纪的“摄影师”如何在没有摄影机的情况下记录下大自然的肌理呢？没错，答案就是蓝晒。蓝晒法（Cyanotype）1842年由英国科学家约翰·赫歇尔爵士发明[1]。他利用柠檬酸铁铵和铁氰化钾的感光特性，创造了这种能生成普鲁士蓝图像的工艺，主要用于复制自己的数学笔记[2]。其关键转折发生在19世纪40年代，英国植物学家安娜·阿特金斯将它变为摄影技术[3]。她制作了《英国藻类照片》——历史上第一本用照片作插图的书籍，通过将植物标本直接放在感光纸上，开创了“光绘图”技法[4-5]。在工业领域，蓝晒长期作为低成本且精准的蓝图复印技术，直到被现代技术取代[6-7]。20世纪90年代后，因其独特的蓝色调，在当代艺术和手工领域复兴[8-9]。

蓝晒的化学原理：
①感光材料构成，蓝晒法主要使用两种化学试剂：柠檬酸铁铵：作为光敏剂，化学式可表示为(NH~4~)~3~Fe(C~6~H~5~O~7~)~2~[10]，对紫外光敏感。铁氰化钾：俗称赤血盐，化学式为K~3~[Fe(CN)~6~][11]，作为显色剂。将两者等比例混合涂布于纸张表面后，纤维素纤维间便均匀负载了上述两种物质，形成感光层。

②光致还原反应，在紫外光（波长300-400 nm）照射下，柠檬酸铁铵中的三价铁离子被还原为二价亚铁离子，柠檬酸根作为电子供体被氧化分解: ![](media/image3.jpeg){width="1.7145833333333333in" height="0.37430555555555556in"}（其中Cit^3-^代表柠檬酸根离子C~6~H~5~O~7~^3-^，氧化产物包括丙酮二羧酸、二氧化碳等小分子）此步骤是光能向化学能转化的核心，生成的二价铁离子是整个显色反应的触发器。

③显色反应，曝光后生成的二价铁离子化学性质活泼，会立即与溶液中的铁氰化钾反应，生成不溶性的深蓝色沉淀——亚铁氰化铁（普鲁士蓝）：![](media/image4.jpeg){width="2.1958333333333333in" height="0.32013888888888886in"}

该生成物的标准化学式为：![](media/image5.jpeg){width="1.18125in" height="0.39166666666666666in"}[12]

式中 x 通常为14-16，表示结晶水数量可变。该物质CAS号为14038-43-8，俗称普鲁士蓝。此沉淀与滕氏蓝（Turnbull's Blue）结构相同，现代晶体学研究已证实两者为同一物质[12]。

④定影原理，最终的水洗过程基于溶解度差异进行选择性分离：曝光区域：生成的不溶性亚铁氰化铁（普鲁士蓝）牢固附着于纸张纤维，形成蓝色图像。未曝光区域：原有的柠檬酸铁铵(Fe^3+^)与铁氰化钾均易溶于水，在流水冲洗中被溶解并脱离基底。通过这种溶解度差异，最终在载体上留下由不溶性普鲁士蓝构成的稳定图像，而空白区域则恢复为基底本色。

### （二）实验方案设计：像侦探一样设计对照实验

#### [1. 探究曝光时间对蓝晒成像效果的影响]

**【实验用品】**
水彩纸、磁吸亚克力板、海绵刷、图像来源 (树叶、羽毛、负片等) 、20%柠檬酸铁铵溶液、10%铁氰化钾溶液、市购的3%过氧化氢溶液（实验中使用配置后的1%过氧化氢溶液）和**6%白醋、小苏打**、蒸馏水、滴管、10 mL小烧杯、托盘、一次性手套、黑色卡纸、自制紫外灯箱。（后续实验中的实验用品一致）

**【猜想与假设】** 曝光时间越长，颜色越深，但会导致细节丢失。

**【设计思路】** 采用“逐格曝光法”，同一张相纸在不同时间揭开，保证其他条件完全一致。

**【实验步骤】**

![](media/image6.png){width="6.464583333333334in" height="0.8729166666666667in"}

1.  **配置感光液**，用滴管分别量取1.5 mL 20%柠檬酸铁铵溶液和10%铁氰化钾溶液，加入烧杯中混合均匀。
2.  **制备感光相纸**，将纸张放入一个干净干燥的托盘中，用刷子蘸取混合液并均匀涂抹在纸张正面（要避免背面蘸上混合液）。
3.  **装片**，将亚克力板、分成16格的不透光黑卡纸和感光相纸从上至下依次叠放，见图1。
4.  **曝光**，将装置放置在阳光下照射，每1分钟抽出一格。
5.  **水洗显影**，取出相纸放入洗液盘中，用自来水冲洗相纸，记录每组成像效果。

#### [2. 探究原料体积比对蓝晒成像效果的影响]

**【猜想与假设】** 两种药液的比例会影响反应程度，从而影响颜色的深浅和层次。

**【设计思路】** 设置多个比例组（2:1，1:1，1:2，1:3），保持总药液量相同，其他条件（纸张、涂刷方式、曝光时间、显影液）相同。

**【实验步骤】**

1.  **配置感光液**，准备四个烧杯分别标注A、B、C、D，按下表用量用滴管分别量取20%柠檬酸铁铵溶液和10%铁氰化钾溶液，加入烧杯中混合均匀。

2.  **制备感光相纸**，将纸张放入一个干净干燥的托盘中，用刷子蘸取混合液并均匀涂抹在纸张正面，每张纸更换新海绵刷，避免交叉污染。
3.  **装片**，将亚克力板和感光相纸从上至下依次叠放。
4.  **曝光**，将装置放置在阳光下照射。
5.  **水洗显影**，取出相纸放入洗液盘中，用自来水冲洗相纸，记录每组成像效果。

#### [3. 探究显影剂选择对蓝晒成像效果的影响]

**【猜想与假设】** 不同的显影液会影响普鲁士蓝的生成或稳定性，从而改变蓝晒成像效果，其中1%双氧水有最明显效果。

**【设计思路】** 选取4种不同显影剂进行对比。

**【实验步骤】**

1.  **配置感光液**，用滴管分别量取1.5 mL 20%柠檬酸铁铵溶液和10%铁氰化钾溶液，加入烧杯中混合均匀。
2.  **制备感光相纸**，将纸张放入一个干净干燥的托盘中，用刷子蘸取混合液并均匀涂抹在纸张正面（要避免背面蘸上混合液）。
3.  **装片**，将亚克力板和感光相纸从上至下依次叠放。
4.  **曝光**，将装置放置在阳光下照射。
5.  **切分**，将相纸平均切分成大小、形状均相同的4份小相纸。
6.  **显影**，将4份相纸分别用4种不同的显影剂（水、1%双氧水、6%白醋、小苏打）显影。
7.  **晾干**，将湿相纸放在室内避光阴凉处干燥，记录每组成像效果。

*# 废液经大量水稀释后排放*

**【安全审查声明】**

本小组在实验方案设计过程中，曾计划使用稀盐酸、草酸探究酸性环境对蓝晒的影响。经指导老师及安全资料核查，我们意识到：铁氰化钾遇强酸会释放剧毒氢氰酸气体[13]，该操作严重违反中学生实验安全规范[14]，绝不可实施。

因此，所有涉及盐酸、草酸及其混合液的实验组别全部删除，改用白醋、柠檬汁等食品级弱酸。在保证科学探究真实性的前提下，将实验安全置于最高优先级。
